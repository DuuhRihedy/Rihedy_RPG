# A Grid / A REDE

## Visão Geral
- **Tipo:** Infraestrutura tecnológica / Rede de comunicação
- **Raridade:** Único (a REDE original) / Raro (cópias)

## Descrição
A **REDE** é uma tecnologia cuja origem e natureza completa são desconhecidas. A **Grid** em [[Gargamorth]] é descrita como uma **"cópia mal feita da REDE"**, cujo servidor geral se encontra no QG Elmet sob controle de [[Lucia Firaga]].

[[Mandamorth]] também possui **The Grid** — possivelmente uma versão diferente ou a mesma rede usada pelas máquinas para controlar a cidade.

## Versões Conhecidas

| Versão | Localização | Controlador |
|---|---|---|
| A REDE (original) | *(Desconhecida)* | *(Desconhecido)* |
| Grid de Gargamorth | QG Elmet | [[Lucia Firaga]] |
| The Grid de Mandamorth | Cidade das Máquinas | Máquinas autônomas |

## Notas do Mestre
- A REDE original é um mistério — tecnologia Elmet? SSA? Algo mais antigo?
- A conexão entre as duas Grids (Gargamorth e Mandamorth) pode ser plot central
- Pode ser resquício da Máquina Vida ou das Sky Pieces
